# GrapheneCS₂₈ Prediction Tool

A Streamlit web application for predicting the 28-day compressive strength (CS₂₈) of graphene-reinforced cementitious composites.

## Features
- **Tab 1:** CS₂₈ prediction using Extra Trees and XGBoost, with real-time SHAP waterfall interpretability
- **Tab 2:** Young's modulus estimation using the physics-informed relationship E = 997.18√CS₂₈
- **Tab 3:** GA-optimised mix designs for nine system–graphene type combinations

## Deployment
1. Fork/clone this repository
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub account
4. Select this repository and set `app.py` as the main file
5. Click Deploy — your app will be live at `https://your-app-name.streamlit.app`

## Dataset
Trained on 127 experimental data points from the graphene-reinforced cementitious composites literature.
